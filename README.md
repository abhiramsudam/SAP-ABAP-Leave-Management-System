# SAP-ABAP-Leave-Management-System
SAP ABAP project for employee leave management
Features are:
- Employee Master Table
- Leave Request Table
- Auto Request Number (SNRO)
- Create Leave Request
- Reject Leave Request
- Fetch Employee Requests
- Status Management

Technologies:
- SAP ABAP
- Dictionary Tables
- Classes & Methods
- Number Range
